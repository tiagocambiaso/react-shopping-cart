# 🍎 Apple Store E-Commerce Cart

A modern, responsive shopping cart application simulating an Apple Store experience. Built from scratch to demonstrate state management, data persistence, and UI/UX best practices.

## 🚀 Tech Stack
* **Frontend Framework:** React 18
* **Language:** TypeScript
* **Styling:** Tailwind CSS
* **Build Tool:** Vite

## ✨ Key Features
* **Dynamic Cart Management:** Add items, modify quantities, and remove products seamlessly.
* **Data Persistence:** Cart state is saved in `LocalStorage`, ensuring data is not lost upon page refresh.
* **Strict Typing:** Comprehensive TypeScript interfaces for products and cart items to prevent runtime errors.
* **Responsive Design:** Mobile-first approach using Tailwind CSS.

## 🛠️ What I Learned
During this project, I strengthened my skills in:
* Managing complex state arrays in React (`useState`, `map`, `filter`).
* Implementing `useEffect` for syncing state with browser storage.
* Refactoring and componentizing UI elements for better readability.
* Handling type safety and naming collisions in TypeScript.
